# Vendor and Manufacturer Manual

## Main tools
- `Vendor Board`
- `Manufacturer Board`

## Purpose
This portal area is used for vendor-side and manufacturer-side participation in bid and RFQ workflows.

## Manufacturer workflow
1. Review open manufacturer RFQs.
2. Open only the work visible to your manufacturer account.
3. Submit or update manufacturer bids.
4. Track bid status changes.

## Internal workflow on the same board
Internal users can:
- create manufacturer RFQs
- review submitted manufacturer bids
- manage the bid process from one routed board

## Visibility rules
- manufacturers should only see their own relevant work
- internal roles can review the broader bid set

## Common mistakes to avoid
- assuming the board shows every RFQ in the system
- submitting bids without verifying job, weight, delivery area, and notes
