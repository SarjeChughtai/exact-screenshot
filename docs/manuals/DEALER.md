# Dealer Manual

## Main tools
- `Dealer RFQ`
- `My Requests`
- dealer profile settings

## Standard workflow
1. Keep the dealer profile current.
2. Submit RFQs through `Dealer RFQ`.
3. Track submitted requests in `My Requests`.

## Dealer RFQ checklist
- client name
- location
- building dimensions
- roof style and pitch
- liners, gutters, insulation, notes

## Best practices
- keep contact email and phone current so operations can follow up
- review past requests in `My Requests` before creating a duplicate submission
- use the dealer portal as the source of truth for dealer-submitted RFQs

## Common mistakes to avoid
- incomplete location information
- duplicate submissions for the same job
- out-of-date dealer profile details
