# Sales Manual

## Main tools
- `Dashboard`
- `Quick Estimator`
- `Estimates Log`
- `Quote RFQ`
- `RFQ Log`
- `Sales Quote Builder`
- `Master Deals`

## Standard workflow
1. Build a quick estimate.
2. Save it or keep it as a draft if work is incomplete.
3. Reopen the estimate from `Estimates Log` when needed.
4. Import the estimate into `Quote RFQ`.
5. Submit the RFQ and confirm it opens in `RFQ Log`.
6. Track estimator and operations progress in the RFQ workflow queue.
7. Build the customer-facing quote once the internal quote is ready.
8. Convert the external quote to a deal when won.

## Daily checklist
- review assigned or owned quotes and RFQs
- confirm client and location fields are complete before RFQ handoff
- use job IDs consistently instead of creating disconnected records
- verify the right RFQ/quote PDF exists before sending anything external

## Key actions
### Quick Estimator
- use `Save Draft` for incomplete work
- use `Save Estimate` for formal estimate history
- use `Convert to RFQ` when ready to hand off

### Estimates Log
- `Edit` reopens an estimate in Quick Estimator
- `Import to RFQ` transfers estimate data to Quote RFQ

### RFQ Log
- use the expanded row to verify RFQ details and attached PDF
- use `Convert to Deal` only once the customer quote is truly won
- use `Revert to Quote` if a deal was created by mistake

## Common mistakes to avoid
- skipping city, province, or postal code before RFQ submission
- converting to deal too early
- creating multiple job records for the same opportunity
- assuming an estimator can see an RFQ if it was never assigned
