# Notion Bridge

This folder is the export-first bridge between `ai-context/` and optional Notion workflows.

## Principles

- `ai-context/` remains the canonical task and memory store.
- `notion/export/` contains derived markdown and JSON artifacts generated by `npm run context:notion:export`.
- `notion-automation` is optional and only used when a compatible MCP connection exists.
- Notion is never treated as the source of truth for task status.

## Generated Outputs

Running `npm run context:notion:export` produces:

- `notion/export/index.md`
- `notion/export/manifest.json`
- `notion/export/tasks-database.json`
- `notion/export/dashboard.md`
- `notion/export/handoff.md`
- `notion/export/decisions.json`
- `notion/export/decisions.md`

## Optional Automation

If you later connect an MCP host compatible with the imported `notion-automation` skill, use the generated exports as the source payloads for page or database updates. The bridge is intentionally export-only by default so the repo still works with no external connections.
